اختبار طلبات API

اختبر REST APIs مثل Postman. يدعم GET و POST و PUT و DELETE و PATCH. يعمل 100% في المتصفح الخاص بك.

ما هي أداة اختبار API؟

أداة اختبار API تسمح لك بإرسال طلبات HTTP إلى أي نقطة نهاية API ورؤية الاستجابات على الفور. هذا ضروري للغاية لـ تصحيح الأخطاء في APIs، اختبار التكاملات، و التطوير. هذه الأداة تعمل مثل Postman لكن مباشرة في المتصفح الخاص بك.

أسئلة شائعة

      
      

        ما هي طرق HTTP المدعومة؟
        
ندعم طرق GET و POST و PUT و DELETE و PATCH و HEAD و OPTIONS. هذا يغطي جميع عمليات REST API الشائعة.

      


      

        لماذا أتلقى أخطاء CORS؟
        
CORS (Cross-Origin Resource Sharing) هي ميزة أمان في المتصفح. إذا لم تسمح API بطلبات من مجالك، ستظهر أخطاء CORS. هذا طبيعي وغير متوقع لأسباب أمنية.

      


      

        هل يمكنني اختبار APIs المحلية؟
        
نعم، طالما أن خادمك المحلي قيد التشغيل ويسمح بطلبات المتصفح. قد تواجه مشاكل CORS مع خوادم التطوير المحلية.

      


      

        كيف أضيف المصادقة؟
        
استخدم علامة التبويب "المصادقة" لإضافة رموز Bearer، أو أضف رؤوسًا مخصصة (Authorization و X-API-Key وما إلى ذلك) في علامة التبويب "الرؤوس".

      


      

        هل بيانتي آمنة؟
        
نعم! يتم إرسال جميع الطلبات مباشرة من متصفحك إلى نقطة نهاية API. لا نقوم بتخزين أو رؤية البيانات أو مفاتيح API الخاصة بك. استخدم نقاط نهاية HTTPS للحصول على أمان إضافي.
